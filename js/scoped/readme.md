# var a = 2
1. 词法分析
词法单元：var,a,=,2

2. 解析
将词法单元转换成一个 逐级嵌套 的程序语法结构树 -- 抽象语法树

3. 生成代码
var a = 2

# 有效标识符
function foo(a){
  var b = 2
  console.log(a+b)
}
a,b是foo的有效标识符

# let vs var
1. var 存在声明提升，let不存在
2. let会和{}形成块级作用域
3. var可以重复声明变量 let不可以
